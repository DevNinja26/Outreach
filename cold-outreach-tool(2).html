API Key — Backend Config

The API key lives in the source code — no login screens, no modals. Open cold-outreach-tool.html in any text editor and update the CONFIG block near the top of the <script> section:

// ── BACKEND CONFIG — edit this block ────────────────── const CONFIG = { apiKey: 'sk-ant-YOUR-KEY-HERE', // ← paste your key model: 'claude-sonnet-4-20250514', // ← swap model if needed maxTokens: 4000, thinkingBudget: 1500 }; // ──────────────────────────────────────────────────────

Get your key at console.anthropic.com. Keep this file in a private GitHub repo — never push a real API key to a public repo.

Profile Data

Your profile is stored in your browser's localStorage. Nothing is sent anywhere except to the Anthropic API during generation.
